Early attempt at making a Music Notation software including a GUI
------------------------------------------------------------------------

PROJECT TITLE: MusicNotation
PURPOSE OF PROJECT: Create a working music notation software with similar functionality to Musescore
VERSION or DATE:
HOW TO START THIS PROJECT: Download BlueJ and open package.bluej in it. Right-click on the Runner class and click on "void main(String[] args)", then click "OK".
AUTHORS: Andrew Wang

Currently displays a rough GUI and an ability to toggle between notes, rests, and staffs. Need to work on implementing changing key signature, adding (storing) notes when input, and playing back stored notes.
